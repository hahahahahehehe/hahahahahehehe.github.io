hexo&github pages搭建个人博客

一、前沿介绍

1.1、简介

  • Github Pages是github官方提供的免费的静态站点托管服务，即通过http协议访问github的静态资源

  • hexo是一个静态博客框架，它是基于node.js运行的，可以将markdown文档渲染成静态的html页面

1.2、整体流程

二、环境搭建

2.1 需安装组件

  1. node.js
  2. git
  3. hexo

2.2 安装node.js

因为hexo是基于node.js环境，而且后续需要使用博客的主题等插件，也可以通过node.js自带的npm很方便的下载安装，所以我们首先得安装node.js环境

2.2.1 进入node.js官网下载

下载之后一直next往下安装就行了

2.2.2 验证安装结果

win+R进入cmd窗口，执行node -v和npm -v,有版本显示，则node.js安装完成

2.3 安装git

2.3.1 进入[git官网](Git (git-scm.com))下载安装

点击next安装完成

2.3.2 验证

进入cmd，执行git --version，输出git版本则表示安装成功

2.3.3 git配置

1、鼠标右键，点击git bash,进入bash界面

2、配置git的客户端的用户名和邮箱

加上–global对整个git客户端生效，不加的话只对当前的项目生效

3、配置本机的公私钥

在git客户端执行命令

然后一直回车，这行完成后会在当前用户家目录c:/User/用户名/.ssh目录下生成公私钥文件

私钥自己流程，公钥复制到github上来进行免密连接

4、登录进入github，点击setting

5、点击SSH and GPG keys,进入ssh设置界面，然后点击new SSH key

将复制的公钥复制到key,Title自己随便填，然后点击Add SSH key就行了

2.4 安装hexo

hexo可以通过npm一键式安装

2.5.1 登录github新建仓库

2.5.2 配置仓库名称

仓库名称要写成github用户名.github.io，这是固定格式，Owner输入框内的就是用户名，比如Owner框里面的是zhangsan,则新建的仓库名必须叫zhangsan.github.io

2.5.3 设置github pages
